## 游戏名称：
popo

游戏玩法：
1.进入游戏后按下回车键或单击鼠标左键开始游戏
2.对于右上角的人物一，上下左右键移动，Ctrl右放炸弹
3.对于左上角的人物二，w s a d 键移动，Ctrl左放炸弹
4.自己放置的炸弹不能炸到自己
5.游戏中的箱子为可被炸开
6.游戏中的树和房子为不可被炸开
7.炸开游戏中的箱子后人物移动速度将会增大
8.场上只剩下一名人物时游戏结束
9.点击鼠标右键或者按Esc、字母O键退出游戏
